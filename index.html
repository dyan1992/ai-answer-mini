API端点文档

Echo API

URL: /api/echo

方法: GET

描述: 返回所有提供的查询参数作为JSON响应。

参数: 任意查询参数

示例:

URL: /api/echo?name=张三&age=25&city=北京
              
响应:
{
  "success": true,
  "echo": {
    "name": "张三",
    "age": "25",
    "city": "北京"
  },
  "timestamp": "2023-07-21T12:34:56.789Z"
}
